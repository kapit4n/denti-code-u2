# Admin Doctors

**Route:** `/admin/doctors`

**Role:** Admin

**Description:** Lists all clinic doctors in a sortable table with columns: ID, Name, Email, Phone, License Number, Office Room Number, and Active/Inactive status badge.

**Key Elements:**
- Sortable doctor table
- Active/Inactive status badges
- Contact and license information
