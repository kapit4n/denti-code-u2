# Admin Settings

**Route:** `/admin/settings`

**Role:** Admin

**Description:** Allows changing the organization's default locale/language. Displays a dropdown of supported locales from the backend and saves the selection.

**Key Elements:**
- Default locale selector
- Supported locales list
- Save preference to backend

- `src/app/admin/settings/page.tsx`
- `src/features/locale/localeApiSlice.ts`
